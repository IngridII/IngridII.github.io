A Digitally Sculpted Head

Statement

This head was one of the first things I’ve ever sculpted using the program Zbrush. I’m familiar with sculpting things using clay but I've never done anything like this digitally. I’m glad that there was some crossover between the two. What is interesting about this piece is that there was no reference used in the creation of it and that there is no one I know with this particular combination of features. This means on some level this is what my mind thinks a generic head and its facial features look like.
